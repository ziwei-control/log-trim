# log-trim - 紫微智控日志修剪工具

> 24 小时监控日志大小，超过 995MB 自动修剪

## 快速开始

```bash
# 安装
bash install-log-trim.sh

# 查看状态
log-trim status

# 检查并修剪
log-trim check
```

## 功能

- ✅ 24 小时不间断监控
- ✅ 超过 995MB 自动修剪
- ✅ 自动备份（保留 5 个）
- ✅ systemd 服务，开机自启
